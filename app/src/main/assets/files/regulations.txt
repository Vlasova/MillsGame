1. Партия начинается с пустого игрового поля. Каждый игрок по очереди выставляет по одной своей шашке на любой из свободных кружков. Как только игрок смог выстроить три свои шашки в ряд по прямой линии, он снимает с доски любую из шашек противника. Этот ряд из трех фишек одного цвета и называется “мельницей”.

2. После того, как игроки выставили все свои фишки на игровое поле, можно начинать ходить.Игроки также по очереди передвигают свои фишки на соседние свободные кружки вдоль линий с целью построить мельницу. Как только у игрока образовался новый ряд из трех фишек, он забирает любую фишку соперника.

3. Если к концу игры у одного из игроков остается только три фишки, он получает преимущество. Во время своего очередного хода игрок может переставлять фишку на любой свободный кружок поля, не зависимо от линий. Его соперник продолжает ходить по старым правилам, пока у него тоже не останется три фишки.

4. Игра считается выигранной, когда противник не сможет построить мельницу: у него осталось только 2 фишки или нет возможности сделать ход.

*Игрок может строить мельницу на одном и том же месте сколько угодно, но фишку соперника снимать за эту мельницу разрешено только один раз.
